To use this to build your own TedX page you need Drush and Drush Make. First, do a:

drush make https://raw.github.com/ramsalt/DrupalMakes/master/dists/drupal-7.7/platform-plain.build

Then clone this git repo into the sites/default folder with: 

git clone git@github.com:ramsalt/tedx.git

..and you are ready to build yopur content types. We will make this a proper install profile in the future, with all configuration bundled in features
